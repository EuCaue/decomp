## Usage

<small><strong>NPM and NodeJS</strong> is required.</small>

1. Install

```sh
npm install @eucaue/decomp -g
```

2. Use

```sh
decomp <FILE_PATH>
```

3. Help command to see all options

```sh
decomp --help
```

## Features

- Supports multiple file types _[.rar, .7z, .zip, .tar, .tar.*]_.
- Output Directory.
- Cross platform.
